### **Namespace can have only 2 resource controls:**

## **1. ResourceQuota**

Controls the **total amount of resources** allowed in the namespace.

You can limit:

* Total CPU for all pods
* Total Memory for all pods
* Total number of Pods
* Total PVC storage
* Total number of Services
* Total number of ConfigMaps / Secrets
* LoadBalancer services count
* GPU limits

ğŸ‘‰ **Think: ResourceQuota = "Total budget for the entire namespace"**

---

## **2. LimitRange**

Defines **default CPU/Memory** or **max/min limits** for each container **inside the namespace**.

You can set:

* Default CPU for containers
* Default Memory for containers
* Max CPU/Memory allowed per container
* Min CPU/Memory required per container

ğŸ‘‰ **Think: LimitRange = "Rules for each container inside this namespace"**

At pod/container level, you only set:

## **1. Requests**

The **minimum** CPU/Memory the container needs.

## **2. Limits**

The **maximum** CPU/Memory the container can use.
